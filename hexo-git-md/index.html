github page 部署hexo博客项目

发表于 2020-03-31 分类于 工具使用指南 阅读次数： Valine：

Hexo博客搭建好了，但是只能本地访问，怎样可以部署到网上并且让其他人访问了。最简单方便的办法就是托管到GitHub上了，这个是
免费托管的，并且还可以绑定自己的域名，不需要自己买服务器搭建，毕竟服务器还是很贵的。所以接下来介绍GitHub上如何部署我们的博客。

注册一个GitHub帐号

怎么注册GitHub帐号就不说了，网上很多教程，可自行搜索。

GitHub新建仓库

步骤如下：

点击用户，选择Your repositories

点击new按钮，新建仓库

Respostory name 中填写yourname.github.io,这里的yourname一定是你的GitHub的用户名
Description随意填，选择Public,然后勾选复选框，点击Create respository 就创建成功了
