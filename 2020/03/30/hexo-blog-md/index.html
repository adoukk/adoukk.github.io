总结

到此hexo的搭建基本就结束了，在此总结一下基本操作

1.编译代码 $ npm install
2.启动服务器 $ hexo server
3.创建分类 $ hexo new page categories
4.新建文章 $ hexo new [article name]
5.删除文章 目录/source/_posts下删除相应的文章，然后执行命令：$ hexo g $ hexo d

另外，如果想在博客中添加评论系统等其他扩展功能，可以去网上找第三方插件即可。参考教程
